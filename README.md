# SAS.3

## Requirements

- Node.js 20+ (required for Vite 5)
- npm 10+ or equivalent package manager

## Setup

```bash
# Check your Node version
node --version

# If using nvm, use the correct version
nvm use

# Install dependencies
npm install

# Start development server
npm run dev
```

## Troubleshooting

If you get a `SyntaxError: Unexpected reserved word` error when running `npm run dev`, your Node.js version is too old. Please upgrade to Node.js 20 or higher.
